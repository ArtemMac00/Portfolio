# Portfolio

## 1. Dispatcher

### Программный помощник диспетчера для управления ремонтным персоналом при авариях на ЛЭП

Система мониторинга погодных условий и оценки рисков для объектов электросетевой инфраструктуры. Помогает диспетчеру быстро оценить обстановку и выдаёт готовые рекомендации для ремонтных бригад. Реализована на примере Вологодской области, легко адаптируется под любой регион.

**Стек:** Python, Flask, VK API, Leaflet, Open-Meteo API, OpenStreetMap, g4f

**Ключевые фичи:**
- Интерактивная карта с цветовой индикацией уровней риска
- Отображение движущихся значков ремонтных бригад
- Чат-бот ВКонтакте с генерацией рекомендаций через нейросеть
- Расчёт риска по 5 факторам с весовыми коэффициентами
- Резервные демо-данные при отказе внешних сервисов
- Автоматическое переключение на локальный анализатор при недоступности ИИ

**Ссылка на GitHub:** [https://github.com/ArtemMac00/Portfolio/tree/main/Dispatcher](https://github.com/ArtemMac00/Portfolio/tree/main/Dispatcher)

## 2. AI Telegram Bot

### Интеллектуальный ассистент для разработчиков

Telegram-бот на базе Agnes AI, который помогает с задачами по Python и программированию. Бот поддерживает контекстный диалог, память на 5 последних сообщений, интерактивные кнопки для управления и считает статистику использования. Подходит для обучения, быстрых справок по коду и автоматизации рутинных вопросов.

**Стек:** Python, python-telegram-bot, Agnes AI API, Requests, python-dotenv

**Ключевые фичи:**
- Контекстный диалог с памятью
- Системный промпт (эксперт по Python)
- Интерактивные кнопки (/reset, /stats)
- Логирование и обработка ошибок
- Бесплатная нейросеть без привязки карты

**Ссылка на GitHub:** [https://github.com/ArtemMac00](https://github.com/ArtemMac00/ai-telegram-bot)
