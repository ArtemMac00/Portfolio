## Проекты

### 1. Dispatcher

### Программный помощник диспетчера для управления ремонтным персоналом при авариях на ЛЭП

Система мониторинга погодных условий и оценки рисков для объектов электросетевой инфраструктуры. Помогает диспетчеру быстро оценить обстановку и выдаёт готовые рекомендации для ремонтных бригад.

**Стек:** Python, Flask, VK API, Leaflet, Open-Meteo API, OpenStreetMap, g4f

**Ключевые фичи:**
- Интерактивная карта с цветовой индикацией уровней риска
- Отображение движущихся значков ремонтных бригад
- Чат-бот ВКонтакте с генерацией рекомендаций через нейросеть
- Расчёт риска по 5 факторам с весовыми коэффициентами
- Резервные демо-данные при отказе внешних сервисов
- Автоматическое переключение на локальный анализатор при недоступности ИИ

**Ссылка на GitHub:** [https://github.com/ArtemMac00/Portfolio/tree/main/Dispatcher](https://github.com/ArtemMac00/Portfolio/tree/main/Dispatcher)

### 2. AI Telegram Bot

### Интеллектуальный ассистент для разработчиков

Telegram-бот на базе Agnes AI, который помогает с задачами по Python и программированию. Бот поддерживает контекстный диалог, память на 5 последних сообщений, интерактивные кнопки для управления и считает статистику использования.

**Стек:** Python, python-telegram-bot, Agnes AI API, Requests, python-dotenv

**Ключевые фичи:**
- Контекстный диалог с памятью
- Системный промпт (эксперт по Python)
- Интерактивные кнопки (/reset, /stats)
- Логирование и обработка ошибок
- Бесплатная нейросеть без привязки карты

**Ссылка на GitHub:** [https://github.com/ArtemMac00/Portfolio/tree/main/AITelegramBot](https://github.com/ArtemMac00/Portfolio/tree/main/AITelegramBot)

### 3. AI Code Reviewer

### ИИ-ревьюер кода с подсветкой, историей и загрузкой файлов

Веб-приложение для анализа кода на Python, C#, JavaScript и Java. Нейросеть находит ошибки, оценивает качество и предлагает оптимизации.

**Стек:** Python, FastAPI, Agnes AI, Chart.js, HTML, CSS, JavaScript

**Ключевые фичи:**
- Анализ кода на 4 языках
- Оценка качества 0–10 с цветовой индикацией
- Поиск ошибок с указанием severity и строки
- Анализ безопасности
- Оптимизированный код с подсветкой синтаксиса
- Загрузка файлов (drag & drop)
- История анализов в localStorage
- Горячие клавиши (Ctrl+Enter)

**Ссылка на GitHub:** [https://github.com/ArtemMac00/code-reviewer](https://github.com/ArtemMac00/Portfolio/tree/main/AiCodeReviewer)

### 4. TaskManager

### Система управления задачами с AI-ассистентом и веб-дашбордом

Полнофункциональная система управления задачами с веб-дашбордом и консольной версией. Помогает организовать работу, отслеживать прогресс и анализировать эффективность с помощью AI-ассистента.

**Стек:** Python, Flask, SQLite, Agnes AI, Chart.js, HTML, CSS, JavaScript

**Ключевые фичи:**
- Веб-дашборд с графиками и статистикой
- Консольная версия с авторизацией
- AI-определение приоритета при создании задачи
- Полный CRUD с фильтрацией и поиском
- Экспорт в CSV и JSON
- Адаптивный дизайн

**Ссылка на GitHub:** [https://github.com/ArtemMac00/TaskManager](https://github.com/ArtemMac00/Portfolio/tree/main/TaskManager)
