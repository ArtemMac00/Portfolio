## О проекте

При разработке на Python часто возникают однотипные вопросы: как написать функцию, почему не работает код, какой алгоритм выбрать. Открывать браузер и искать ответ каждый раз — неудобно и отвлекает.

Данный проект — это Telegram-бот, который:

- Отвечает на вопросы по Python и программированию
- Пишет и объясняет код с примерами
- Помнит контекст диалога (до 5 последних сообщений)
- Даёт рекомендации по структуре и оптимизации кода
- Работает на бесплатной нейросети Agnes AI
- Управляется через интерактивные кнопки

Бот написан на Python с использованием библиотеки `python-telegram-bot` и API Agnes AI. Может быть легко адаптирован под другие языки программирования или тематики.

## Установка и запуск

### Требования

- Python 3.9 или выше
- pip (менеджер пакетов)
- Токен Telegram-бота (получить у [@BotFather](https://t.me/botfather))
- API-ключ Agnes AI (получить на [platform.agnes-ai.com](https://platform.agnes-ai.com/))

### 1. Клонирование репозитория

```bash
git clone https://github.com/ArtemMac00/ai-telegram-bot.git
cd ai-telegram-bot
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

Файл requirements.txt:

```
python-telegram-bot>=20.0
requests>=2.28.0
python-dotenv>=1.0.0
```

### 3. Настройка переменных окружения

Создай файл .env в корне проекта:

```env
TELEGRAM_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
AGNES_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
MODEL=agnes-2.0-flash
```

### 4. Запуск бота

```bash
python bot.py
```

При успешном запуске в консоли появится сообщение:

```
🚀 Запуск бота...
🤖 Модель: agnes-2.0-flash
🔗 API: https://apihub.agnes-ai.com/v1
💬 Жду сообщений...
```

Бот готов к работе. Напиши ему в Telegram любое сообщение.

## Как работает бот

При отправке сообщения бот выполняет следующие шаги:

1. Получает сообщение от пользователя
2. Добавляет его в историю диалога
3. Формирует запрос к Agnes AI с учётом:  
   · Системного промпта (роль эксперта по Python)  
   · Истории диалога (последние 5 сообщений)
4. Получает ответ от нейросети
5. Сохраняет ответ в историю
6. Отправляет ответ пользователю с кнопками управления

### Системный промпт

Бот использует следующую системную роль:

«Ты — опытный Python-разработчик и ИИ-ассистент. Отвечай кратко, понятно и по делу. Если пользователь просит код — приводи примеры с пояснениями. Если вопрос не по программированию — отвечай вежливо, но старайся помочь.»

Это позволяет боту давать качественные и релевантные ответы.

### Лимиты Agnes AI

· 20 запросов в минуту — бесплатно  
· Модель: agnes-2.0-flash — быстрая и умная  
· Без привязки карты — полностью бесплатно

## Надёжность

В системе предусмотрены механизмы устойчивости к сбоям:

· При недоступности Agnes AI бот возвращает понятное сообщение об ошибке  
· Все ошибки записываются в лог с указанием времени и причины  
· Внешние запросы имеют таймаут 30 секунд, чтобы не зависать  
· Автоматическая обработка ошибок через error_handler  
· Корректная обработка длинных сообщений (разбивка на части)
