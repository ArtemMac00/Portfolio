# 🤖 AI Telegram Bot

**ИИ-ассистент в Telegram** с поддержкой контекстного диалога на базе Agnes AI.

---

## ✨ Возможности

- 🧠 **Умные ответы** на вопросы по Python, программированию и IT
- 💬 **Память диалога** — бот помнит контекст беседы
- 🎨 **Интерактивные кнопки** для удобного управления
- 📊 **Статистика** использования
- 🔒 **Безопасное хранение** токенов через `.env`
- 📝 **Логирование** всех действий
- ⚡ **Автоматическая обработка ошибок** — бот не падает

---

## 🛠 Технологии

- **Python 3.9+**
- **python-telegram-bot** — работа с Telegram API
- **Agnes AI** — бесплатная нейросеть для ответов
- **python-dotenv** — управление переменными окружения
- **Requests** — HTTP-запросы к API

---

## 🚀 Установка и запуск

### 1. Клонируй репозиторий
git clone https://github.com/ArtemMac00/ai-telegram-bot.git
cd ai-telegram-bot

### 2. Установи зависимости
pip install -r requirements.txt

### 3. Создай файл .env
Скопируй .env.example и заполни своими токенами:   
TELEGRAM_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz   
AGNES_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx   
MODEL=agnes-2.0-flash

### 4. Запусти бота

## 🎯 Команды бота

Команда	Описание   
/start	Главное меню с приветствием   
/reset	Очистить историю диалога   
/stats	Показать статистику использования   
/help	Помощь и список команд

## 🔧 Конфигурация

Все настройки хранятся в config.py:

Параметр	Значение	Описание  
MAX_HISTORY_LENGTH	10	Количество пар вопрос-ответ в памяти  
MAX_TOKENS	500	Максимальная длина ответа  
TEMPERATURE	0.7	Креативность ответов (0.0 — строго, 1.0 — творчески)  
SYSTEM_PROMPT	Текст роли	Что бот говорит о себе
