## О проекте

TaskManager — это полнофункциональная система управления задачами с веб-дашбордом и консольной версией. Проект включает:

- **Веб-дашборд** с графиками и статистикой
- **Консольную версию** для быстрого доступа
- **AI-ассистента** на базе Agnes AI для определения приоритетов
- **Управление задачами** (CRUD) с фильтрацией и поиском

---

## Возможности

### Веб-дашборд
- 📊 **Статистика** — общее количество, в работе, выполнено, просрочено
- 📈 **Графики** — распределение по приоритетам, активность за неделю
- ✅ **Управление задачми** — добавление, изменение статуса, удаление
- 🔍 **Поиск** по названию и описанию
- 🧠 **AI-определение приоритета** — автоматически или вручную
- 🎨 **Адаптивный дизайн** — работает на всех устройствах

### Консольная версия (CLI)
- 💻 **Полный CRUD** — создание, чтение, обновление, удаление
- 🧠 **AI-определение приоритета** — автоматически или вручную
- 📂 **Экспорт** в CSV и JSON
- 🎨 **Цветной вывод** с эмодзи
- ⏰ **Уведомления** о просроченных задачах
- 🔐 **Авторизация** с регистрацией пользователей

## Установка и запуск

### Требования
- Python 3.9 или выше
- API-ключ Agnes AI ([platform.agnes-ai.com](https://platform.agnes-ai.com/))

### 1. Клонирование репозитория
```bash
git clone https://github.com/ArtemMac00/TaskManager.git
cd TaskManager
```

### 2. Установка зависимостей

```bash
cd backend
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

```bash
cp .env.example .env
```

Открой .env и вставь свой API-ключ Agnes AI:

```env
AGNES_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
MODEL=agnes-2.0-flash
```

### 4. Запуск сервера

```bash
python app.py
```

Сервер запустится на http://0.0.0.0:5000

### 5. Открытие веб-интерфейса

Открой frontend/index.html через Live Server в VS Code или любой другой локальный сервер.

### 6. Запуск консольной версии

```bash
cd ../cli
python main.py
```
